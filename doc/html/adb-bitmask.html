Detailed Description

The BitMask<T> class is a templat eclass that provides iteration over set bits.

It is a wrapper around std::bitset<T> and lets you iterate over 0-indexed positions of the set bits in a value T. For example:


  int16_t value = 0b0001000110001000;
  adb::BitMask<int16_t> mask(value);

  for(int pos : mask)
      std::cout << pos << ' ';

  \\Prints: 3 7 8 12

The primary usage is to iterate over results of adb::match().
