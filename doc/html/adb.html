int adb::match(char byte, const char *data)

Performs comparison between byte and data and returns int representing matching positions.

The data will be interpreted as a 16 bytes array. The result of the comparison is stored in lowest 16 bits of the return value.

Example:


  const char *data = "aaabcccbdddbeeeb";
  const char value = 'b';
  int result = match(value, data);
  //result == 0b00000000000000000001000100010001;

Note: Requires SIMD (Single Instruction Multiple Data) to be supported by your compiler.
