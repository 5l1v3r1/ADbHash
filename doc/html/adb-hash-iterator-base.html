Detailed Description

The iterator_base<ValueType, ReferenceType, HashType> is base class for STL style iterators for adb::Hash.

It lets you iterate over elements of the adb::Hash.

In order to create a const or non-const iterator from iterator_base you need to provide three template arguments:

  • ValueType
  • ReferenceType
  • HashType

See adb::Hash::iterator and adb::Hash::const_iterator.

The iterator_base is a bidirectional iterator.
